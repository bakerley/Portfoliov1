Findway - 2019

Findway is Saas platform providing careers coach and trainees all the tools to book, follow and help the trainees find the best careers advices possible. A youtube video (in French) featuring the pitch and key functionalities of the website is available below.

Access the code Presentation

About the mission

This project was part of the coding bootcamp program. I pitched the idea and designed the platform, and, with a team of four, developed, implemented and shipped to production the platform, over a two week intense coding sprint. Key features included:

  • Login / Sign-up
  • Possibility for the user to filter and select a coach.
  • Creating the booking system based on a calendar.
  • Implemented an internal chat
  • Give a review after a meeting on the coach
  • Blog section

Technical details

  • Developed typical CRUD routes for the view and administration of projects protected by authorization rights (Pundit & Devise gems).
  • Designed the website mostly using Bootstrap, Javascript ES6 and external JS modules, bundled with Webpack.
  • Made both visitor and admin sides of the website as dynamic as possible through the use of AJAX requests and ActionCable
  • Created the asynchrone Chatrooms
  • Allowed the user to create their availability with a drag and drop calendar
  • Implemented external services for the Chat rooms and Calendar system.
  • Handled the deployment of the app with Heroku and the domain name setup (SSL).

Key technologies

Ruby on Rails 5.2 API JS ES6 Webpack Bootstrap PostgreSQL Heroku

Bon Terroir - 2020

Bon Terroir is a side project that I created with a friend at the time of the first covid lock-down. The projects aims to help the local productors to sell their products online in a really simple and effective way.

Website

About the Project

This project was initiated with a friend during the covid lockdown. We understood that most of the small local productor didn't have a website or an easy way to sell their products. We decided to create this marketplace in order to let the locals create their online shop and continue to do business for the time of the lockdown. ** We put the project on hold as I moved to Canada and my friend as an other job.**

Key features included:

  • Login/Sign up
  • Personnal dashboard and metrics for the vendors
  • Search and filter options for the products
  • Checkout process
  • Stripe integration

Technical details

  • Set up the Shopify account
  • Theme integration
  • Stripe setup
  • Implemented external services modules in order to take care of the shipping

Shopify JS ES6 Stripe SEO
