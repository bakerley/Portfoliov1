Food2Wine - 2021

Food2Wine is a web app application that finds the perfect wine that matches with your food. The user inputs an image and retreive a receipe, a list of groceries and 4 wines that matches the meal. A youtube video featuring the pitch and key functionalities of the website is available below.

Access the code Presentation Slides

About the mission

This project was part of the coding bootcamp program. We pitched the idea and designed the platform, and, with a team of two, developed, implemented and shipped to production the platform, over a two week intense coding sprint. Key features included:

  • User can choose an input (link, image or text)
  • The user gets a recipe.
  • The user gets a grocery list.
  • The user gets four wine recommendation

Technical details

  • Cleaned and transformed the datasets using pandas and stopwords from nltk.
  • Trained a gensim Word2Vec model in 300 vectors embeddings.
  • Computed cosine-distance in order to match the embeddings to a 7 vectors taste palette to predict the best wines.
  • Implemented inverse cooking pretrained model from facebook AI to get the recipes.
  • Implemented the streamlit app.
  • Created the python package with Docker.
  • Deployed the package with Google cloud.
  • Datasets: Amazon fine food reviews and winemag-data-130k from Kaggle

Key technologies

Python 3.8 Pytorch Word2Vec Streamlit Docker Google Cloud Platform
WhatWaste - 2021

The idea of this project was to create a image classifier to recognize the waste in order to throw it in the right trash.

Access the code Access the app

About the project

This is a personnal project. I have train a VGG16 model with custom layers in order to classify a picture into 6 categories. The categories represents the main categories of recycling goods. From my personnal experience, I never know in which trashcan to throw the waste and if it is possible to recycle! Key features included:

  • User inputs a picture of his waste.
  • The model predicts the type of waste.
  • The user get as output the class, recommendations and docs.

Technical details

  • Built custom dataset from Google Open Image v6.
  • Trained a custom VGG16 convolutional neural network in order to class the waste.
  • The trained model has an accuracy of 69.6%.
  • Created an app with Flask.
  • Created the container with Docker.
  • Deployed the container with Google cloud.

Key technologies

Python 3.8 Tensorflow Flask Docker GCP
